# Further Resources & Reading

Companion resources for *101 Data Science Drawings* and broader data science learning.

---

## From the Author

- **TikTok:** [@MinuteData](https://www.tiktok.com/@minutedata) — short visual explainers of data science concepts
- **YouTube:** [@MinuteData](https://www.youtube.com/@MinuteData) — longer walkthroughs and tutorials

## Books

- **An Introduction to Statistical Learning (ISLR)** — James, Witten, Hastie, Tibshirani. The standard accessible introduction to machine learning. Free PDF available at [statlearning.com](https://www.statlearning.com/).
- **Storytelling with Data** — Cole Nussbaumer Knaflic. Practical guide to data visualisation for communication.
- **Causal Inference: The Mixtape** — Scott Cunningham. Accessible econometrics for causal inference. Free at [mixtape.scunning.com](https://mixtape.scunning.com/).

## Online Courses

- [Khan Academy: Statistics and Probability](https://www.khanacademy.org/math/statistics-probability) — Foundations for beginners
- [fast.ai: Practical Deep Learning](https://www.fast.ai/) — Hands-on machine learning with a top-down approach
- [Mode SQL Tutorial](https://mode.com/sql-tutorial/) — Interactive SQL learning from basics to advanced

## Tools & Platforms

- [Kaggle](https://www.kaggle.com/) — Datasets, competitions, and notebooks for practice
- [Google Colab](https://colab.research.google.com/) — Free hosted Jupyter notebooks with GPU access
- [SQLite Online](https://sqliteonline.com/) — Browser-based SQL practice environment
- [Flourish](https://flourish.studio/) — No-code data visualisation tool
